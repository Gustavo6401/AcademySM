# Como Contribuir para a Academy SM?

Existem diversas formas de contribuir para a Academy SM, dentre elas sugerir ou desenvolver novas features, apontar bugs e até mesmo compartilhar esse repositório.

Vale lembrar que você é obrigado a concordar com os termos contados diretamente no nosso [CLA](CLA.md) para que você possa contribuir com código na Academy SM.
