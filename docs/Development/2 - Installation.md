# Instalação

## O que eu preciso para Rodar essa aplicação?

Alguns serviços são necessários para o funcionamento da Academy SM, aqui em baixo, colocarei um link para cada um deles.

[.NET SDK](https://dotnet.microsoft.com/en-us/download)
[Node JS](https://nodejs.org/en/download/package-manager)
[SQL Server Management Studio](https://learn.microsoft.com/pt-br/sql/ssms/download-sql-server-management-studio-ssms?view=sql-server-ver16)
[Mongo DB Community Server](https://fastdl.mongodb.org/windows/mongodb-windows-x86_64-8.0.3-signed.msi)
[Mongo DB Compass](https://downloads.mongodb.com/compass/mongodb-compass-1.44.5-win32-x64.exe)
[Visual Studio Community](https://visualstudio.microsoft.com/pt-br/thank-you-downloading-visual-studio/?sku=Community&channel=Release&version=VS2022&source=VSLandingPage&passive=false&cid=2030)

**Observação**
Precisamos que você Instale duas cargas de trabalho, mais especificamente a carga de Desenvolvimento Web com .NET e desenvolvimento Web com Node JS.
