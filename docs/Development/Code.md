# Como Funciona o Código na Academy SM?

Desenvolvemos a Academy SM utilizando principalmente 3 APIs e uma aplicação React para servir como Front-End. Na Visual Studio, temos a estrutura de pastas própria da IDE que contém 4 diretórios virtuais, sendo eles: Client, Server, DevelopmentUtils e Tests.

Vamos falar um pouco sobre cada um dos projetos?

1. **academysm-client**: Nosso projeto principal e que tem mais código atualmente utilizado na Academy SM, é o Front-End escrito utilizando o VITE, o React e o Node JS. Misturei aqui código TypeScript e código JavaScript, afim de melhorar a minha experiência de desenvolvimento.
2. **CadastroUsuario**: API responsável por prover informações como os dados de cadastro e gerar as contas do usuário.
3. **Groups**: API principal e responsável pelos dados dos grupos.
4. **FileServer**: API cuja responsabilidade é receber os arquivos enviados pelos usuários.

Falarei individualmente de cada uma dessas APIs.
