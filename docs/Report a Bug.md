# Como eu Relato um Erro que Está Acontecendo?

Relatar bugs é um processo simples, porém, ele exige um conhecimento básico em Markdown, a mesma linguagem de marcação que você pode utilizar para editar mensagens no WhatsApp.

Você pode clicar na aba de Issues do Github:

![Issues](<imgs/Instrução Issues.jpeg>)

Após fazer esse processo, é necessário que você clique em "Nova Issue", conforme mostrado na imagem:

![Tela de Issues](<imgs/Tela de Issues.jpeg>)

Clique em Bug Report:

![Bug Report](<imgs/Bug Report.jpeg>)

Aqui você pode Relatar o Bug:
